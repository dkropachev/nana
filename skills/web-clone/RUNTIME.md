---
name: web-clone
description: Compact runtime contract for URL-driven site cloning
---

<Purpose>
Clone a target webpage with structure, styling, and core interaction parity.
</Purpose>

<Use_When>
- The user says `web-clone`, `clone site`, `clone website`, or `copy webpage`.
- The task is a live URL-driven visual/functional clone.
</Use_When>

<Runtime_Rules>
- Use browser tooling to extract the source page before implementation.
- Treat visual and functional verification as required.
- Keep scope to the provided page unless the user asks for more.
</Runtime_Rules>
