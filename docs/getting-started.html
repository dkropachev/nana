Getting Started

End-to-end setup for first-time users.

Prerequisites

  • OpenAI Codex CLI installed: npm install -g @openai/codex.
  • Codex authenticated before the first nana launch.
  • Git (recommended) for project workflows.

Installation

npm install -g @openai/codex

curl -L -o nana <release-binary-url>
chmod +x nana
sudo mv nana /usr/local/bin/nana

nana setup
nana doctor

Recommended Launch

nana                   # standard launch with configured defaults
nana --high            # optional one-off reasoning override
nana --madmax          # trusted environments only
nana --fast            # issue Codex /fast before the rest of the prompt

nana --madmax is an explicit opt-in that bypasses approvals and sandboxing, so reserve it for trusted environments.

First Run Walkthrough

  1. Create or enter a project directory.
  2. Run nana setup to install prompts, skills, AGENTS.md, and configuration.
  3. Start with nana, or use nana --high for a one-off reasoning override.
  4. Use nana --fast if you want NANA to issue Codex /fast first, and reserve nana --madmax for trusted environments because it bypasses approvals and sandboxing.
  5. Use $deep-interview "clarify the auth change" when intent or boundaries are still unclear.
  6. Use $ralplan "approve the auth plan and review tradeoffs" to lock the implementation plan.
  7. Use /skills to browse the installed helpers inside Codex when you want to see the available workflow surfaces.
  8. Continue with direct implementation, nana review <pr-url>, or nana work start <issue-or-pr-url> based on what you are doing next.

Prompt Instruction Source

NANA layers project instructions by default (extends Codex behavior, does not replace/bypass Codex core system policies):

-c model_instructions_file="<cwd>/AGENTS.md"

Override behavior when needed:

NANA_BYPASS_DEFAULT_SYSTEM_PROMPT=0 nana
NANA_MODEL_INSTRUCTIONS_FILE=/path/to/instructions.md nana

Troubleshooting

IssueWhat to check
nana command not foundReinstall the release binary and ensure its install location is in PATH.
codex command not foundInstall Codex CLI with npm install -g @openai/codex and confirm the global bin directory is in PATH.
Prompts not availableConfirm files exist in ~/.codex/prompts/ and re-run nana setup.
Skills not loadingVerify ~/.codex/skills/*/SKILL.md files were installed.
Doctor reports config issuesRun nana doctor and apply the suggested fixes.
